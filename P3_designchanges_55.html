3. Design Changes

We have done some changes, after both feedback from our supervisor, and some things we found out in the process of coding.

Changes to frontpage.html

Initially we did not think about having a footer with contact information and opening hours, but we got some inspiration from other websites and then we found out we wanted to have footer including this. It makes the site more user-friendly and nice. We removed the social media icons we had and put it in the footer. We made an own html page called footer.html. We did the same for the topbar, and called it topbar.html. We also included 5 pictures in the front page, instead of 3-4. Other than that, the front page is as similar to how we imagined it would be.

Changes to openinghours.html

We centered the opening hours instead of having it on the same line as the map.

Changes to course.part and topbar.part

Changes to topbar.js, frontpage.js, courses.js, information.js

We decided to make only one JavaScript file instead of several. We wanted to compile it into one file to have fewer HTTP requests. We wanted to make as few requests to the server to improve the performance of the page. We called the JavaScript file script.js.

Changes to style.css

We have one main css for the different pages, but we decided to make one external stylesheet for the topbar and footer. We thought this would make it a bit more organized and clean.

Additional

  • We have added footer.html and topbar.html as mentioned above.
  • When discussing the top bar, we decided that when scaling the site, the original top bar should dissappear and a hamburger icon/button appears. The hamburger button includes the different alternatives on the top bar. Our logo will float to the left side of the page while the rest of the top bar goes inside the hamburger button.
  • We added some pictures for the front page called front_image1,2,3,4,5.png
  • Furthermore, we added a avocado favicon on our page, called favicon.png
  • We also added pictures of the logo and the social media buttons
